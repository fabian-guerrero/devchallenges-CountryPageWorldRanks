# 🌍 Country Page WorldRanks

A production-ready, responsive web application built with **React** and
**Vite** that visualizes global country rankings and detailed country
information.

This project was developed as a solution to the _Country Page --
WorldRanks_ challenge from DevChallenges, with a strong focus on clean
architecture, scalability, performance, and maintainability.

## 🧠 Project Overview

Country Page WorldRanks provides:

- A structured visualization of country ranking data
- A detailed country information view
- Responsive layouts across mobile and desktop
- Clean component architecture following modern React best practices

The application emphasizes:

- Separation of concerns
- Reusable UI components
- Scalable folder structure
- Performance-aware rendering
- Clean and maintainable code

---

## 🏗️ Architecture & Design Principles

This project follows:

- **Component-driven architecture**
- **Separation of UI and data logic**
- **Modular folder structure**
- **Reusable and composable components**
- **Single responsibility principle**
- **Declarative React patterns**
- **Scalable styling strategy**

The structure allows easy feature extension, testing integration, and
migration to larger-scale applications.

## 💻 Getting Started

### Prerequisites

- Node.js 16+
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/fabian-guerrero/devchallenges-CountryPageWorldRanks.git

# Navigate to the project directory
cd devchallenges-CountryPageWorldRanks

# Install dependencies
npm install

# Start development server
npm run dev
```

Open your browser at:

    http://localhost:5173

---

## 📦 Production Build

```bash
npm run build
```

Production-ready files will be generated in the `dist/` directory.
